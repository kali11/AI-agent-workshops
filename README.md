# AI-agent-workshops
A repository with labs for AI agent workshops. 

More details:
https://ai72h.piotrkalinowski.cloud/

All labs are self-paced. You can do them in any order however I recommend to start with 01_Langchain_basics / Lab 1.

### Workshops content:

1. 01_Langchain_basics
    - Lab 1 - building chains using LCEL (Langchain Expression Language)
    - Lab 2 - building LinkedIn post generator
    - Lab 3 - building chatbot
2. 02_Vector database & RAG
    - Lab 1 - Embeddings and loading documents into vector database
    - Lab 2 - building RAG (Retrieval Augmented Generation)
    - Lab 3 - building RAG agent
3. 03_AI_agents
    - Lab 1 - creating first agent
    - Lab 2 - building mulit-tools agent
    - Lab 3 - HOMEWORK
4. 04_Multi-agent systems
    - Lab 1 - building multi-agent system with AutoGen
    - Lab 1 - building multi-agent system with Langgraph
